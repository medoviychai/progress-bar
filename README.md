# ProgressBar Component

Input props: 
1. array like this:
  [
    {name: 'Sold', color: '#BD1FBE', value: 677},
    {name: 'Got free', color: '#FC64FF', value: 23},
    ...
  ]
2. height - number, bar line height
3. width - number, bar line width
